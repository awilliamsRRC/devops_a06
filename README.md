# Simple Banking Backend with Student Portfolio Frontend

This repository houses a simple banking backend application built using Flask, designed to demonstrate core banking functionalities. It is complemented by a student portfolio frontend application, which includes a link to access the banking system.

## Features

**Banking Backend (Flask):**
* User account management (creation, login)
* Deposit functionality
* Withdrawal functionality
* Balance inquiry


**Student Portfolio Frontend:**
* Showcase for student projects and information.
* Direct link to the banking backend application for interaction.
